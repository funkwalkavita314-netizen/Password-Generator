🔐 CLI Password Generator (Python)

 This is a simple and secure Command Line Interface (CLI) based Password Generator built using Python.

The program allows users to generate strong and random passwords based on their preferred length and security level.

🚀 Features

 Custom password length selection

 Multiple security options:
1. Only Letters
2. Letters + Numbers
3. Letters + Numbers + Special Characters
 
 Random and secure password generation
Input validation with error handling

Repeat option to generate multiple passwords


🛠 Technologies Used:

1. Python
2. random module
3. string module

💡 Purpose of Project

This project was built to strengthen understanding of:
1. Loops
2. Conditional statements
3. String manipulation
4. Exception handling
5. User input handling
